## FinSightAi – Agentic Investment Insights

FinSightAi is an AI-powered financial intelligence dashboard for investment bankers, traders, and research teams. It helps you compare companies, explore industries, and generate pitch‑ready insights using clean visualizations and agentic workflows.

## Tech stack

- Vite + React + TypeScript
- shadcn-ui + Tailwind CSS

## Local development

```sh
npm install
npm run dev
```

The app will be available on `http://localhost:8081` (configured in `vite.config.ts`).

## Production notes

- Favicon and social cards use the FinSightAi logo (`public/finsightai-logo.svg` / `finsightai-logo-og.svg`).
